# Sistema Integral para la Gestión y Operación de una Empresa Organizadora de Eventos

## Descripción

Sistema web y móvil diseñado para digitalizar y automatizar los procesos operativos y administrativos de una empresa organizadora de eventos.

La plataforma permitirá centralizar la gestión de clientes, eventos, cotizaciones, pagos, recursos y proveedores, mejorando la eficiencia operativa, el seguimiento de actividades y la experiencia de los clientes.

---

## Objetivo

Desarrollar una solución tecnológica alojada en la nube que permita administrar de forma integral los procesos de una empresa organizadora de eventos mediante una plataforma moderna, escalable y segura.

---

## Problemática

Actualmente muchos procesos relacionados con la organización de eventos se realizan de forma manual o mediante múltiples herramientas aisladas, lo que genera:

- Dificultad para gestionar clientes y eventos.
- Seguimiento limitado de pagos y contratos.
- Falta de control sobre recursos y personal asignado.
- Gestión ineficiente de proveedores.
- Duplicidad de información.
- Poca visibilidad del estado general de las operaciones.

## Funcionalidades Principales

### Gestión de Eventos y Servicios

- Catálogo de eventos.
- Catálogo de servicios.
- Administración de paquetes.
- Consulta de información detallada.

### Gestión de Clientes

- Registro de clientes.
- Edición de información.
- Historial de eventos contratados.
- Historial de servicios.
- Seguimiento de contratos.
- Historial de pagos.
- Consulta de cotizaciones.

### Cotizaciones

- Creación de cotizaciones.
- Consulta de cotizaciones previas.
- Impresión y exportación de documentos.
- Asociación con eventos y clientes.

### Pagos

- Registro de pagos.
- Generación de tickets.
- Historial de transacciones.
- Planes de pago.
- Control de saldos pendientes.

### Recursos y Personal

- Registro de organizadores.
- Asignación de personal a eventos.
- Gestión de actividades.
- Seguimiento de tareas.

### Gestión de Proveedores

- Registro de proveedores.
- Clasificación por categorías.
- Control de disponibilidad.
- Consulta de costos y servicios.
